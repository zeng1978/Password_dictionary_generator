# Password Dictionary Generator v4.0

高级密码字典生成器｜Hashcat 掩码 + 字典合并 + 文件分割 + 多线程 + GUI

## 项目简介

Password Dictionary Generator v4.0 是一个面向密码安全研究、渗透测试与
Hashcat
用户的高级密码字典生成工具，支持掩码生成、字典合并、字符处理、性能优化等功能。

## 功能特性

-   Hashcat 掩码生成
-   字典 A/B 合并
-   字符大小写转换、反转、移除、添加
-   自定义字符集组合
-   多线程写入
-   自动文件分割
-   性能监控
-   GUI 图形界面（Tkinter）

## 安装

``` bash
pip install -r requirements.txt
```

## 运行

``` bash
python Password_dictionary_generator_v4.0.py
```
